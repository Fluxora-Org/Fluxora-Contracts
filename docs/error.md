# Error Code Reference Table

This table lists all possible errors in the FluxoraStream contract, including both formal `ContractError` variants and runtime errors/panics that can occur. It enables integrators and frontends to handle errors correctly.

| Error Code / Panic Message                 | Description                                                                          | Functions Returning It                                                                                                                                                             |
| ------------------------------------------ | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `StreamNotFound` (1)                       | The specified stream does not exist                                                  | `pause_stream`, `resume_stream`, `cancel_stream`, `withdraw`, `calculate_accrued`, `get_stream_state`, `cancel_stream_as_admin`, `pause_stream_as_admin`, `resume_stream_as_admin` |
| `StartTimeInPast` (5)                      | `start_time` is before the current ledger timestamp; creation rejected               | `create_stream`, `create_streams`                                                                                                                                                  |
| `StreamAlreadyPaused` (10)                 | Stream is already in `Paused` state; redundant pause rejected                        | `pause_stream`, `pause_stream_as_admin`                                                                                                                                            |
| `StreamNotPaused` (11)                     | Stream is not `Paused`; cannot resume an `Active` stream                             | `resume_stream`, `resume_stream_as_admin`                                                                                                                                          |
| `StreamTerminalState` (12)                 | Stream is `Completed` or `Cancelled` (or past `end_time`); modification blocked      | `pause_stream`, `resume_stream`, `pause_stream_as_admin`, `resume_stream_as_admin`                                                                                                 |
| `Unauthorized` (6)                         | Caller is not authorized to perform this operation                                   | `init` (bootstrap admin auth), `set_admin`, `require_stream_sender`, admin overrides                                                                                               |
| `InvalidState` (2)                         | Operation attempted in an invalid state (e.g., withdraw from non-terminal paused)    | `cancel_stream`, `withdraw`, `withdraw_to`, `batch_withdraw`                                                                                                                       |
| `InvalidParams` (3)                        | Function input parameters are invalid (e.g., withdraw to contract address)           | `create_stream`, `withdraw_to`                                                                                                                                                     |
| `ContractPaused` (4)                       | Global pause active; creation blocked until admin calls `set_contract_paused(false)` | `create_stream`, `create_streams`                                                                                                                                                  |
| `InsufficientBalance` (8)                  | Token transfer failed due to insufficient balance or allowance                       | `create_stream`, `cancel_stream`, `withdraw`                                                                                                                                       |
| `InsufficientDeposit` (9)                  | Deposit amount does not cover the planned duration at the specified rate             | `create_stream`, `create_streams`                                                                                                                                                  |
| `AlreadyInitialised` (7)                   | Contract has already been initialized                                                | `init`                                                                                                                                                                             |
| `can only close completed streams`         | Stream must be Completed to be closed                                                | `close_completed_stream`                                                                                                                                                           |
| `contract not initialised: missing config` | Contract storage not initialized before access                                       | `get_config`, `get_token`, `get_admin`                                                                                                                                             |
